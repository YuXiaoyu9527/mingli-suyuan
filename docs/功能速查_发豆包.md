## 一、今日首页（打开App第一眼）

### 1. Calendar Hero（顶部大字区域）
- 展示今天日柱天干（如"丙"字，超大字号）
- 根据当前24节气自动变色：春=苍岭绿、夏=朱砂红、秋=秋麦金、冬=黛蓝
- 标注当前节气和五行（如"🌾 芒种·五行属火"）
- 下面显示农历日期+星期+生肖

### 2. 宜忌卡片
- 左半：宜（绿色字，如"嫁娶·祭祀·入学"）
- 右半：忌（红色字，如"动土·出行"）
- 来源：《协纪辨方书》规则+lunar-python每日宜忌数据

### 3. 四象四宫格（2x2工具入口）
每个格子有：图标+名称+描述+四象方位标注+顶部五行色带
- 🔮 周易占卜 | 朱雀·南·火
- 💑 合婚配对 | 青龙·东·木
- ✏️ 八字起名 | 白虎·西·金
- 🏠 阳宅风水 | 玄武·北·水

### 4. 每日古籍（随机推送）
- 从4800+条古籍库中随机选一条展示
- 标注来源（如"《三命通會》·卷七"）
- 点击可展开全文弹窗

### 5. 历史命例（随机展示）
- 每天随机展示一位历史人物的八字
- 如：左宗棠/曾国藩/李白/苏轼/康熙等
- 显示姓名+八字+格局+可信度（正史记载/野史传闻）
- **点击自动跳转排盘页并填入该人物生辰数据**

---

## 二、排盘页（核心功能）

### 输入区
- 公历/农历一键切换（农历输入→后端lunar-python自动转公历）
- 年月日时分精确输入（5列网格）
- 性别选择 + 出生城市（用于真太阳时校正）
- "从亲友档案库导入"快捷入口（读取本地已存档案）

### 空白期（未排盘时）
- 展示经典命例推荐卡片（2x2网格）
- 10位历史人物中随机展示4个（司马光/范蠡/诸葛亮/苏轼/李白/康熙/岳飞/王阳明/张居正/武则天）
- 每格显示emoji+姓名+格局描述
- 点击一键填入排盘表单

### 排盘结果区
**四柱展示**：年柱/月柱/日柱/时柱，每柱天干地支各一字
- 八个字用**莫兰迪五行色**分别着色：木=苍岭绿、火=朱砂红、土=琥珀黄、金=秋麦金、水=黛蓝
- 日柱有红色光环标记

**摘要行**：日主+五行+格局名称+旺衰判断+用神/忌神推荐

**三个结果Tab**：
- 概览：格局分析文字 + 用神分析文字 + 五行分布条（5色进度条）
- 详情：每柱天干地支十神藏干长生等详细信息 + 地支六合六冲三刑六害关系
- AI解读（需DeepSeek API Key）：AI基于八字+古籍引用生成简批

**底部引流区**：
- 💑 合婚配对 → 跳转合婚页（带入当前八字参数）
- ✏️ 八字起名 → 跳转起名页（带入当前八字参数）
- 分享卡片（文字版八字摘要）
- 免责声明："仅供传统文化研究与民俗参考，不构成人生决策依据"

## 三、周易占卜页

### 起卦环节
- 问题输入框 + 5个预设快捷问题
- **两种起卦方式选择**（二选一，不是直接开始）：
  - 🕐 时间起卦：以年月日时数字起卦，含体用生克分析，适合问时机
  - 🎲 随机起卦：模拟三枚铜钱投六次，适合问运势决策
- 选中后点"开始起卦"

### 起卦动画（视觉爆点！）
- ☯️ 太极旋转 1.8秒
- 💥 金光一闪
- 六条爻线从底部依次砸入（弹簧物理动画）
- 变爻用红色圆圈标注
- 卦名从模糊→清晰渐变

### 结果展示（三个分层卡片）
**① 卦象解读**：白话解释本卦含义+变卦含义+与你问题的关联

**② 行动建议**：根据你的具体问题类型（事业/感情/财运/学业/健康/综合）给出行动步骤，不是通用模板

**③ 五行调和·具体举措**（核心差异化！）：
- 穿衣颜色建议（如"红色、紫色、玫红"）
- 随身物品推荐（如"南红玛瑙、香薰蜡烛"）
- 具体行动清单（如"中午11-13点打电话""在南方位放一盏暖灯"）
- 有利方位

**占卜记录**：历史问卦列表

## 十、全局设计亮点（视觉向，拍视频必提）

| 亮点 | 具体表现 |
|---|---|
| 宣纸纹理背景 | 暖米色底色+纤维纹理渐变，像古籍纸张 |
| 鎏金分割线 | 卡片间渐变金色细线 |
| 去煞圆角 | 所有卡片统一rounded-2xl（16px圆角），不刺眼 |
| 莫兰迪五行色 | 5种低饱和度古典色，不用荧光亮色 |
| 交错入场动画 | 卡片从下往上逐个弹入（stagger效果） |
| PWA支持 | 浏览器打开→添加到桌面→像原生App一样使用 |
| iOS安装引导 | 自动检测iOS，弹出添加到主屏幕的操作指引 |

## 十一、技术架构（可提但不用细讲）

- 前端：Next.js 16 + React 19 + Tailwind CSS v4 + Framer Motion
- 排盘引擎：lunar-python（100%代码计算，不依赖AI）
- AI解读：DeepSeek-V4
- 古籍检索：ChromaDB向量搜索 + SimpleSearcher关键词保底
- 部署：Netlify（前端已上线）+ Railway（后端待上线）
