# 源码打包清单 — Gumroad ZIP

---

## 打包命令（在 Git Bash 里执行）

```bash
cd "d:/DAI_Factory_Workspace/AI新项目/创业项目"

# 创建打包目录
mkdir -p ../destinyscroll-v1.0

# 复制需要的文件（排除 node_modules / pycache / .next / chromadb）
rsync -av \
  --exclude='web/node_modules' \
  --exclude='web/.next' \
  --exclude='__pycache__' \
  --exclude='*.pyc' \
  --exclude='.git' \
  --exclude='data/chroma_db' \
  --exclude='.env' \
  --exclude='docs/录制清单_照着做.md' \
  --exclude='docs/抖音拍摄脚本_v0.3.md' \
  --exclude='docs/功能速查_发豆包.md' \
  --exclude='docs/superpowers' \
  --exclude='work' \
  --exclude='reference' \
  --exclude='个人情况分析与建议总结.md' \
  ./ ../destinyscroll-v1.0/

# 打包成 ZIP
cd ..
# Windows 用户：右键文件夹 → 发送到 → 压缩文件夹
# 或者用 PowerShell:
# Compress-Archive -Path destinyscroll-v1.0 -DestinationPath destinyscroll-v1.0.zip
```

## 打包后检查

```
□ ZIP 大小应该在 5-15 MB（不含 node_modules 和 chroma_db 就不会太大）
□ README_EN.md 是买家解压后第一个看到的文件
□ .env.example 存在（让买家知道需要什么环境变量）
□ 没有 .env 文件（避免泄露你的 API Key）
□ 没有 .git 文件夹
□ 没有个人分析文档
```

## 最后一步：上传到 Gumroad

1. 打开 gumroad.com → 你的产品页
2. Content → Upload file → 选择 destinyscroll-v1.0.zip
3. 设置版本号 v1.0
4. Save → 产品就上架了
