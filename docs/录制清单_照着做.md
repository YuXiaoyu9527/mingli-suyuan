# 抖音素材录制清单 · 照着做就行

> 🎯 目标：录完全部素材，拿去剪辑就能用
> ⏱️ 预计耗时：手机录屏 20 分钟 + 电脑录屏 10 分钟 + 真人出镜 10 分钟
> 📁 全部录完后，在电脑上建一个文件夹 `抖音素材/` 把文件放进去

---

## 一、准备：开录前先做这些

```
□ 手机关闭所有通知（飞行模式+开WiFi，避免弹窗）
□ 手机开启「显示触摸操作」（设置→辅助功能→触摸→显示触摸）
□ 手机亮度调到 70% 以上
□ 清理桌面图标，用纯色壁纸
□ 准备木质/素色桌面作为俯拍背景（可选）
□ 电脑打开 VSCode，加载好创业项目
□ 电脑浏览器打开 Gemini 对话页面，滚动到"去煞圆角/三才步长"那段
□ 电脑终端 cd 到项目目录，准备好 git log --oneline
```

---

## 二、手机录屏部分（主素材—20分钟）

> 录制工具：iPhone 控制中心自带录屏 / 安卓用 AZ Screen Recorder
> 每条素材录 15-30 秒，**动作要慢**，手指在每个位置停 1 秒再动

### 素材 A：今日首页全景（10秒）
```
操作：打开 App → 首页完全加载后 → 手指从顶部慢慢滑到底部 → 停留2秒 → 结束
要拍到的内容：Calendar Hero（节气+大字）→ 宜忌卡片 → 四宫格 → 古籍 → 命例
镜头感觉：像在"欣赏"，不是"操作"
```
**重点**：录的时候确保大日柱显示的颜色是节气色（6月=朱砂红），这就是"节气变色"的素材。

### 素材 B：四宫格特写（8秒）
```
操作：在首页停住 → 手指依次指过四个宫格（朱雀→青龙→白虎→玄武）
     每个格子停留 1 秒，让观众看清四象标注和五行色带
镜头感觉：像在"指给朋友看"
```

### 素材 C：排盘—输入生辰（8秒）
```
操作：点排盘Tab → 展示公历/农历切换（点一下农历再切回来）
     → 手指点在"年"输入框 → 一个个修改数字（改成 1811）
     → 手指标注城市输入框（不用真填）
镜头感觉：展示精确度
```

### 素材 D：排盘—经典命例一键填入（6秒）
```
操作：在排盘页空白期 → 手指点一个经典命例（比如苏轼）→ 表单自动填入
     → 手指点"排盘"按钮
要拍到的：命例卡片 → 表单填入 → 按钮点击 的完整链条
```

### 素材 E：排盘结果—四柱五色特写（12秒）⭐ 核心素材
```
操作：排盘结果出来后 → 手指慢慢从年柱滑到月柱→日柱→时柱
     → 然后手指停在一个字上（比如甲木=苍岭绿）
     → 镜头可以凑近一点（手机靠近屏幕）
要拍到的：八个字五种颜色，日柱有红色光环
镜头感觉：像在看一幅画，慢、稳
```
**这是全App最好看的画面，多录几遍，选最稳的一条。**

### 素材 F：排盘结果—格局用神+五行条（8秒）
```
操作：在"概览"Tab → 手指从格局分析滑到用神分析 → 滑到五行分布条
要拍到的：格局名称（如"正官格"）、用神推荐（如"用神：木火"）、5色进度条
```

### 素材 G：周易占卜—完整流程（20秒）⭐ 核心素材
```
操作：
1. 在首页点占卜 → 输入问题"8月份能成单吗"
2. 手指在"时间起卦"和"随机起卦"之间犹豫一下
3. 点「随机起卦」
4. 等待太极旋转 → 金光一闪 → 六条爻线砸入（这段不要说话，让动画自己跑）
5. 卦名显现后，手指往下滑
6. 滑到「五行调和」卡片 → 手指逐一点过：穿衣颜色、随身物品、具体行动
```
**这是最炸裂的素材，从起卦到五行调和，录完整的 20 秒。**

### 素材 H：古籍搜索（10秒）
```
操作：在首页点古籍检索 → 点热门搜索标签"正官格"
     → 结果出来后手指点第一条
     → 标注来源：《三命通會》·卷七
要拍到的：搜索框→结果→古籍来源标注
```

### 素材 I：历史命例→跳排盘（8秒）
```
操作：在首页滑到底部 → 点历史命例卡片（比如左宗棠）
     → 自动跳转排盘页，表单已填入
     → 手指点一下已填入的数据
要拍到的：命例卡片 → 跳转 → 表单自动填入
```

### 素材 J：学堂答题（12秒）
```
操作：点学堂Tab → 选一个章节 → 点开始答题
     → 故意选错一个答案 → 弹出AI导师+古籍引用
     → 手指点"错题本"入口
要拍到的：课程列表→答题→答错→AI答疑→错题本
```

### 素材 K：我的—亲友档案（8秒）
```
操作：点我的Tab → 手指点"新增" → 输入姓名"曾国藩" → 保存
     → 点刚创建的档案 → 跳转排盘页
要拍到的：档案创建→列表→一键跳排盘
```

### 素材 L：PWA添加到桌面（6秒）
```
操作：在浏览器打开App → 底部弹出安装横幅 → 手指点"安装"
要拍到的：横幅弹出→点击安装→桌面出现图标
```

## 三、电脑录屏部分（AI证据素材—10分钟）

> 录制工具：Windows 用 Xbox Game Bar（Win+G）/ Mac 用 QuickTime
> 所有素材在 VSCode 和终端里录

### 素材 M：CLAUDE.md 滚动（15秒）
```
操作：
1. 打开 VSCode → 打开 CLAUDE.md 文件
2. 用鼠标滚轮从上往下慢慢滚
3. 在「AI 使用红线」那一段停 2 秒
4. 在「玄学 UI 隐性规范」那一段停 2 秒
5. 滚到底部
要拍到：完整的项目宪法文件，红线标注部分停留够久
```
**对应的文案**："这是我的项目宪法，AI 每次开工前必须读它。里面写死了——AI 不碰排盘计算、每句话必须标古籍出处。"

### 素材 N：git log 滚动（10秒）
```
操作：
1. 打开终端 → cd 到项目目录
2. 输入 git log --oneline（不要回车）
3. 录屏开始后，回车执行
4. 让画面停留 2 秒，展示 30 次提交记录
5. 用鼠标从上往下逐行指过几个关键 commit：
   - "init"（第一条）
   - "feat: UI redesign Phase 1"
   - "feat: paipan upgrade + zhouyi revamp"
   - "feat: 玄学排版具象化"
要拍到：完整的提交历史，手指/鼠标点几个关键版本号
```
**对应的文案**："30 次提交，从一行代码到完整 App。每一行都是 AI 写的。"

### 素材 O：Gemini 对话截图（10秒）
```
操作：
1. 打开浏览器 → Gemini 对话页面
2. 找到"去煞圆角/三才步长/洛书九宫"那一段
3. 录屏：从你的提问开始慢慢往下滚
4. 在 Gemini 回复的"去煞圆角"那行停 1 秒
5. 在"三才步长"那行停 1 秒
6. 在"莫兰迪色系"那行停 1 秒
要拍到：你和 AI 讨论设计的完整对话
```
**对应的文案**："连界面设计都是 AI 给的方案——去煞圆角、三才步长、莫兰迪色系。我一个都不懂，但 AI 懂。"

### 素材 P：项目文件树展开（8秒）
```
操作：
1. VSCode 左侧文件树 → 从顶部逐层展开
2. web/src/app/ → 展开所有页面文件夹（paipan/zhouyi/xuetang/mingli...）
3. api/paipan/ → 展开所有 Python 引擎文件
4. data/ → 展开古籍数据
5. 鼠标在文件树上慢慢划一圈
要拍到：庞大的文件结构
```
**对应的文案**："11 个页面、8 个 Python 引擎、4800 条古籍数据。一行代码都没亲手写。"

## 四、真人出镜部分（5-10分钟）

> 设备：手机前置摄像头 + 三脚架，或找人帮你拍
> 衣服：纯色（藏青/米白/黑），不要花纹和 logo
> 背景：干净墙面/书架/木质桌面
> 光线：面朝窗户，自然光从正面或侧面来
> 表情：自然说话，不要念稿

### 需要真人出镜的画面（共 4 段，每段 5-10 秒）

#### 出镜 1：开头亮相（5秒）
```
位置：画面左侧或中间，面对镜头
台词："我因为想系统学八字和周易知识，
       但市面上没有好用的学习工具。
       所以零基础借助 AI，花 15 天做了这个。"
注意：说"15天"时可以用手指比一下，增加肢体语言
```

#### 出镜 2：中间转折（5秒）
```
位置：同出镜1
台词："我不懂代码。这些全部是对着 AI 说人话，
       AI 帮我写出来的。"
注意：说"我不懂代码"时可以笑一下，显得真实
```

#### 出镜 3：展示初心（5秒）
```
位置：同出镜1
台词："这个 App 不收费、不算命。
       就是我自己的学习笔记，分享给同好。"
注意：说"不收费"时停顿半秒，让观众消化
```

#### 出镜 4：结尾引导（5秒）
```
位置：同出镜1，可以稍微侧身
台词："后续我会分享更多 AI 做项目的经验。
       对传统文化或 AI 编程感兴趣的，关注我。"
注意：结尾语气要笃定，不要犹豫
```
