## 三、拍摄小贴士

### 录屏建议
- iPhone 用自带录屏（控制中心 → 录屏）
- 安卓用 AZ Screen Recorder
- **关键**：录屏前开启「显示触摸操作」（让观众看到手指点哪里）
- 关闭通知，避免弹出消息打断

### 视觉加分项
- 手机壳用素色（推荐米色/木纹/纯黑），呼应App设计
- 桌面背景用纯色/极简，不要花花绿绿
- 如果真人出镜：穿素色中式衬衫（藏青/米白），画面质感立刻拉满
- 手如果出镜：指甲干净，背景简洁

### 节奏控制
- 前 3 秒决定划不划走 → 必须有"钩子"（大字/悬念/对比）
- 中间每 5-8 秒换一个画面 → 信息密度要高
- 结尾必须有 CTA（评论区/链接/关注）

### 诚实策略
- **不要说"完全体"** → 说"v0.3，还在迭代中"
- **不要隐藏后端未部署** → 说"目前需要本地跑后端，我自己也在等 Railway 部署"
- **把"不完美"变成"真实"** → 独立开发者的人设就是"边做边改"
- **观众更愿意关注"正在成长的产品"** 而不是"已经完成的完美产品"

### 发布策略
- 视频1（快剪引流）→ 首条发布，测试流量
- 视频2+3（功能+对比）→ 隔天发布，保持热度
- 视频4（创作者故事）→ 周末发，长尾流量
- 每条视频评论区置顶：「链接在主页/进群获取」
- 回复每一条评论（尤其是问"怎么用"的）
