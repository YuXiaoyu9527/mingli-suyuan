# 命理溯源 — 项目宪法

## 产品定位
基于古籍数字化 + RAG + 精准排盘的中国传统命理学习与考据工具。
不做恐吓式算命，不改运收费。

## 技术栈
- 前端：Next.js 16 + React 19 + Tailwind CSS v4 + Framer Motion + TypeScript
- 后端（后续）：FastAPI + Python 3.13
- 排盘引擎：lunar-python（主），不碰 sxtwl（Windows 编译太痛）
- 向量库：ChromaDB
- 嵌入模型：BGE-M3
- LLM：DeepSeek-V4（OpenAI 兼容 API）
- 部署：Vercel（前端）+ Railway（后端）
- 目标形态：PWA，移动端浏览器打开，可添加到桌面

## 🔴 AI 使用红线（触犯即推翻）

1. **AI 绝不计算干支、排盘、推算大运、起神煞。**
   排盘 100% 由代码（lunar-python）完成。AI 只做解读和教学。
   
2. **每句古籍引用必须包含 `original` 字段（原文）和 `source` 字段（出处）。**
   出处格式：`《书名》·卷X`。无法标注出处的内容不得输出。

3. **无匹配时说"经籍未载此格，不敢妄断"。**
   禁止编造古籍内容或虚构历史人物八字。

4. **禁止绝对断语。**
   禁止"你一定……""你命里……"等句式。改为"古籍认为……""传统命理中……"。

5. **所有历史命例必须标注可信度**：`正史记载` / `野史传闻` / `后人推定`。

6. **免责声明必须出现**：所有解读页面底部标注"仅供传统文化研究与民俗参考，不构成人生决策依据"。

## 🟡 开发规范

1. 排盘结果必须可交叉验证（与问真八字 App 对比）
2. 择吉规则必须代码实现，不靠 AI 记忆
3. 相同输入 → 相同输出（排盘/择吉结果完全确定性）
4. 所有文案写死"传统命理认为""古籍记载"，不写"你会""
5. 界面文案使用传统文化口吻，但不搞神秘主义
